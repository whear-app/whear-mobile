# Quick Start Guide

## Exact Commands to Run

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Start Expo
```bash
npx expo start
```

### Step 3: Choose Platform
- Press `i` for iOS Simulator
- Press `a` for Android Emulator  
- Scan QR code with Expo Go app

## That's It!

- ✅ Compile and run
